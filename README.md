# Lashilangri
基于拉格朗日插值法的找规律题目秒杀器

*极致的屎山，造就狗屎函数* ;
**x^6**写成**xxxxxx**，**6x**写成 **(x)6**;
平均七项式长度：**10k+**;
**0鲁棒性** 压制性的屎!
